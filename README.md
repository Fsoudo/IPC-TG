## Sobre o Projeto

O **EasyMed** é uma aplicação Android que auxilia pacientes na gestão da toma semanal de medicamentos. O objetivo é proporcionar uma interface simples, intuitiva e acessível, com foco em utilizadores com necessidades específicas — idosos e cuidadores familiares.
